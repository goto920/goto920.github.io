1. Summary

  1. Manuals
  2. Installation is NOT necessary
    • Download data size: less than 2MB
    • Network connection is NOT required once downloaded until the application is closed. On most devices, the application can be invoked from local cache.
    • iOS is the exception, network connection is required again after the application is closed. (20180127 version supports off-line use while the app is in background task.)
  3. Supported devices (JavaScript enabled browser in general)
    • Web browsers: Firefox, MS Edge, Google Chrome, Safari etc.
      MS IE(Win10) did not work for me.
    • Operating Systems: Ubuntu(Linux, my PC), Android Phone(SO02H), iOS 11(I bought a used iPod for this), Windows10(Surface Pro3)、macOS(mac Mini, latest macOS)
  4. Features
    • Standard metronome(incl. odd time signature patterns)
    • Combination of cowbells, conga, etc.
    • 200+ groove drum patterns (sound samples are of my drumkit)
    • Voice count: one, two, ... eight (Female voice samples are fake; just modified male(my) voice samples. Replaced by free samples on gotsound.org.)
    • Tempo adjustment by 0.1bpm step: Tap, Slider, +- buttons
    • Practice timer: in seconds or bars
    • Advanced playback control (for drummers): Swing(Swung) adjustment, automatic bpm incre/decrement, random bar(s) mute, even clicks mute
    • Multiple set lists: with preset clicks/drum pattern or custom loop at specified bpm
    • Custom loop: with preset clicks/drum patterns
  5. Omitted features
    • Creation/editing of preset patterns
    • Polyrhythm
    • Simultaneous playback of clicks and drum patterns
    • Others I did not feel necessary
  6. Background
    • JavaScript programming practice for my work
    • I am an amateur drummer and wanted to include special features for drummers in this program.
    • Wanted to write a generic program for any OS and Web browser. (I know user interface is not as good as apps for each platform.)
    • I will use this program everyday. Bug fix support will be continued until summer of 2018 or until I get bored.
  7. Source code(open source) https://github.com/goto920/nodejs/(metronome)
What's next: Programs for adding clicks to the music (including pre-count), evaluation of time for drum practice, and so on..
